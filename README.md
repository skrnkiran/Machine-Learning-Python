# Machine-Learning-Python
The project involves conducting network and sentiment analysis of Reddit fortnite gaming brand accounts and their followers.
This project explores community behaviour and sentiment within Reddit Fortnite-related gaming accounts by combining network analysis, natural language processing, and machine learning.
Using Python, the project scrapes public Reddit data, analyses user interaction patterns, builds follower/engagement networks, and evaluates sentiment expressed towards brands, updates, and community discussions.
